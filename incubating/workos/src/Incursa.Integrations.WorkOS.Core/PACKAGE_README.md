# Core Source Folder

This is a source folder compiled into `Incursa.Integrations.WorkOS` (not a standalone package).

## Contents

- Deterministic API key authentication result modeling.
- Permission-to-scope mapping and authorization evaluation services.
- API key management lifecycle helpers (create/list/get/revoke orchestration).
- WorkOS webhook signature validation and event processing coordination.
- OAuth client-credentials token provider with cache/refresh behavior.
- Claim enrichment services for organization, role, and permission projection.
