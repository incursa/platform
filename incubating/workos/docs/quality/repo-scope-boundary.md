# Integration Scope Boundary

This repository owns WorkOS-specific integration libraries released independently from platform primitives.

## In Scope

- WorkOS auth, API key lifecycle, webhook verification, and organization mapping integration layers.
- ASP.NET Core wiring and persistence adapters specific to WorkOS scenarios.

## Out Of Scope

- Platform primitives and provider-agnostic orchestration (owned by `platform`).
- Non-WorkOS provider integrations.
