# Integration Scope Boundary

This repository owns Cloudflare-specific integration libraries released independently from platform primitives.

## In Scope

- Cloudflare KV, R2, custom hostnames, and load-balancing clients.
- Cloudflare-specific DI extensions, options, and runbooks.

## Out Of Scope

- Platform primitives and provider-agnostic orchestration (owned by `platform`).
- Non-Cloudflare provider integrations.
