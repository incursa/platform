# Incursa.Platform.Webhooks.AspNetCore

`Incursa.Platform.Webhooks.AspNetCore` provides endpoint and hosted-processing integration for `Incursa.Platform.Webhooks` in ASP.NET Core.

## Install

```bash
dotnet add package Incursa.Platform.Webhooks.AspNetCore
```

## What You Get

- Endpoint mapping extensions for webhook ingress
- Hosted processing integration for queued webhook handling
- ASP.NET Core-friendly registration and options patterns

## Typical Use

Use this package in ASP.NET Core services to expose webhook endpoints and run webhook processing workers in the same host.
