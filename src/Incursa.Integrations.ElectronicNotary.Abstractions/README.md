# Incursa.Integrations.ElectronicNotary.Abstractions

This folder contains the shared-contract package for the Electronic Notary integration family.

Use it when you need a stable vendor-specific dependency between the base package, proof client, and ASP.NET Core hosting surface.

## Where It Fits

- below application code that composes the Electronic Notary integration
- below the proof and ASP.NET Core packages in this vendor family
- outside the provider-neutral capability layer
